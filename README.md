# Lumiq — Responsive Landing Page

A fully responsive landing page for **Lumiq**, a fictional focus productivity app. Built with pure HTML5 and CSS3 using Flexbox for layout.

---

## 📁 Project Structure

```
Task_2/
├── index.html       # Main HTML file
├── style.css        # External stylesheet
└── README.md        # Project documentation
```

---

## 📄 Pages / Sections

| Section | Description |
|---|---|
| Header / Navbar | Sticky navigation with logo and CTA button |
| Hero | Headline, subtext, CTA buttons and app mockup |
| Social Proof | Trusted company logos bar |
| Features | 6 feature highlight cards |
| Testimonials | 3 user review cards on dark background |
| CTA Band | Call-to-action banner |
| Footer | 4-column footer with links |

---

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Styling and animations
- **Flexbox** — Responsive layout
- **Google Fonts** — Syne (display) + Inter (body)

---

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| Desktop (> 900px) | 3-column features, side-by-side hero |
| Tablet (≤ 900px) | 2-column features, stacked testimonials |
| Mobile (≤ 680px) | Single column, hamburger menu shown |
